# Task 3 – Explore Coding & Collaboration Platforms

**Module:** 3
**Marks:** 20
**Role:** Student Digital Ambassador

---

## Overview

This task involved exploring coding practice platforms and Google Workspace
collaboration tools to build familiarity with platforms used in academic
and professional environments.

---

## Part A – Coding Practice Platform

**Platform Used: HackerRank**

I created an account on HackerRank using my college email and completed the
beginner-level "Solve Me First" challenge, which introduced the basic structure
of solving coding problems on the platform. This gave me hands-on exposure to
how competitive coding environments work and how to read problem statements,
write solutions, and submit code for automated evaluation.

---

## Part B – Google Workspace Collaboration

**Tool Used: Google Forms**

I created a 5-question Digital Literacy Awareness Quiz using Google Forms,
designed to test my batchmates' understanding of key digital literacy concepts
including safe internet practices, collaboration tools, digital footprints,
and email etiquette. The form includes multiple choice and short answer questions
and is linked to a Google Sheet for live response tracking.

## Reflection

Exploring HackerRank and Google Forms reinforced how different platforms serve
distinct academic and professional purposes. HackerRank helped me understand
structured problem-solving in a competitive coding environment — a skill directly
relevant to technical interviews and internships in cybersecurity and software
development. Google Forms, on the other hand, demonstrated how cloud-based tools
enable real-time collaboration and data collection without any infrastructure
setup. As a Cyber Domain Lead at KAVACH, I can see myself using both platforms
actively — HackerRank for sharpening technical skills and Google Forms for
running awareness campaigns and quizzes within my community.
