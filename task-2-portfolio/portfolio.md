# Task 2 – Build Your Student Digital Portfolio

**Module:** 2
**Marks:** 20
**Role:** Student Digital Ambassador

---

## Overview

This task involved setting up a professional digital presence across key platforms used by employers, internship coordinators, and the academic community.

---

## Platforms Chosen

### 1. GitHub
A version control and code hosting platform used by developers worldwide. I created a profile README repository (named after my username) that includes my name, branch, year, institution, and a learning goal. I plan to use GitHub over the next four years to host projects, contribute to open source, and showcase my coding journey.

### 2. LinkedIn
A professional networking platform used by recruiters and academics. I filled in my Education section with my current degree, institution, and expected graduation year. Over the next four years, I plan to grow my network, share achievements, and use LinkedIn to find internship and placement opportunities.

### 3. Kaggle
A data science and machine learning competition platform. I created an account to explore datasets and participate in beginner challenges. Over the next four years, I plan to complete Kaggle courses, earn certificates, and build a data science portfolio through competitions.

---

## Reflection

Setting up these three platforms gave me a clearer picture of what a professional digital identity looks like at an early stage. One thing I found interesting was how each platform serves a distinct purpose — GitHub for technical work, LinkedIn for professional networking, and Kaggle for domain-specific learning. The most challenging part was writing a concise GitHub README that feels personal yet professional. This task made me realise that building a digital presence is an ongoing process that starts from day one of college.
